# Static Archive

هذا المجلد يحتوي ملفات ثابتة قديمة نُقلت من جذر المشروع أثناء التنظيم.

## المحتوى

- `styles/`: ملفات CSS قديمة لم تعد مرتبطة مباشرة بالصفحات الحالية.
- `components/`: ملفات مكونات JavaScript قديمة.
- `pages/`: ملفات JavaScript قديمة خاصة بصفحات منفصلة.

## الاستخدام

هذه الملفات محفوظة كمرجع فقط. مسار التشغيل الحالي يعتمد على:

- `assets/css/site.css`
- `assets/js/main.js`
- `assets/js/page-shell.js`
- `assets/js/site-config.js`
- `data/site.js`

إذا احتجت استرجاع جزء من الأرشيف، افحص المراجع أولًا ثم انقل الجزء المطلوب فقط إلى المسار النشط المناسب.
